Azexo Reveals

Version 1.0

How to use

Files:
- easejs.js and tweenjs.js - canvas animation library;
- azexo_transitions.js - animation library;
- azexo_reveals.js - utilities to simplify of using library (on scroll handle, access to library via classes and attributes).

Classes:
- azexo-reveal: determine that it will be reveal effect;
- azexo-circle-(1,2,3,4,5): determine circle effect number;
- azexo-plane-(1,2,3,4,5): determine plane effect number;
- azexo-inner/azexo-outer: used only for circle effect to determine of maximum radius (inner or outer radius of element box).

Attributes:
- data-azexo-color-1: determine color used in some effects (by default used CSS "color" style);
- data-azexo-start-delay: delay after on scroll event before animation start;
- data-azexo-angle: direction of planes moving (only for planes effect).

Restrictions:
- Transparency emulated via solid-color which correspond to "background-color" property of parent DOM element;
- Element box should not be overflowed by visible content. Animation will show only inside of element box so all overflowed visible content will be clipped.

Custom effect
Instead of using of azexo-circle-(№)/azexo-plane-(№) classes you can make your custom effect via data-azexo-circles/data-azexo-planes attributes. This attributes should contain JSON string with array of objects (circles or planes).

Properties of circle object:
- inside(default: false): direction of circle animation;
- color(default: background color): filling color of circle;
- compositeOperation(default:'source-over'): canvas draw mode;
- beforeDelay(default: 0): delay in milliseconds before start animation of this circle;
- afterDelay(default: 0): delay in milliseconds after animation of this circle before it will be hide;
- duration(default: 1000):duration of animation in milliseconds;
- ease(default:'quintOut'): available functions "linear", "quadIn", "quadOut", "quadInOut", "cubicIn", "quintOut", "cubicInOut", "quartIn", "quartOut", "quartInOut", "quintIn", "quintInOut", "sineIn", "sineOut", "sineInOut", "backIn", "backOut", "backInOut", "circIn", "circOut", "circInOut", "bounceIn", "bounceOut", "bounceInOut", "elasticIn", "elasticOut", "elasticInOut";
- alpha(default: 1): drawing transparency;
- radius(default:outer radius): maximum radius;
- x(default: center of box): x coordinate of circle center;
- y(default: center of box): y coordinate of circle center.

Properties of plane object:
- inside(default: true): position of plane along direction of animation (inside box of element or not);
- color(default: background color): filling color of plane;
- compositeOperation(default:'source-over'): canvas draw mode;
- beforeDelay(default: 0): delay in milliseconds before start animation of this plane;
- afterDelay(default: 0): delay in milliseconds after animation of this plane before it will be hide;
- duration(default: 1000): duration of animation in milliseconds;
- ease(default:'quintOut'): available functions "linear", "quadIn", "quadOut", "quadInOut", "cubicIn", "quintOut", "cubicInOut", "quartIn", "quartOut", "quartInOut", "quintIn", "quintInOut", "sineIn", "sineOut", "sineInOut", "backIn", "backOut", "backInOut", "circIn", "circOut", "circInOut", "bounceIn", "bounceOut", "bounceInOut", "elasticIn", "elasticOut", "elasticInOut";
- alpha(default: 1): drawing transparency;
- rotate(default: 0): direction of plane animation in degrees.

Example:
class="azexo-reveal"
data-azexo-circles='[
{
    "beforeDelay": "0",
    "duration": "0",
    "radius": "outer",
    "afterDelay": "2000"
}, {
    "beforeDelay": "0",
    "color": "red",
    "duration": "1000",
    "radius": "inner",
    "afterDelay": "600"
}, {
    "beforeDelay": "300",
    "duration": "1000",
    "radius": "inner",
    "afterDelay": "300"
}, {
    "beforeDelay": "900",
    "duration": 1000,
    "radius": "inner",
    "compositeOperation": "destination-out",
    "afterDelay": "1000"
}
]'
